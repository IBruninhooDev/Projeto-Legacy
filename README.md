Website oficial do servidor **Legacy RP**, focado em proporcionar a melhor experiência de Roleplay no MTA (Multi Theft Auto). Este projeto serve como portal para a comunidade, oferecendo informações sobre o servidor, loja, galeria e links oficiais.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica de todas as páginas.
- **CSS3**: Estilização moderna com variáveis, flexbox, grid e animações personalizadas.
- **JavaScript (Vanilla)**: Lógica para menu mobile, carregamento de página (loader) e interatividade.
- **Google Fonts**: Tipografia "Inter" para um visual limpo e profissional.

## 📂 Estrutura de Arquivos

O projeto está organizado da seguinte forma:

```text
/
├── index.html          # Página inicial (Hero section, destaque e introdução)
├── sobre.html          # Informações detalhadas sobre o servidor e regras
├── loja.html           # Portal para doações e aquisição de itens VIP
├── galeria.html        # Exibição de capturas de tela e momentos do servidor
├── ip.html             # Página dedicada à conexão e informações técnicas do servidor
├── links.html          # Central de links externos (Instagram, YouTube, Discord)
└── src/                # Pasta de recursos do sistema
    ├── css/
    │   └── style.css   # Estilos globais e componentes da interface
    ├── js/
    │   └── script.js  # Scripts para comportamento dinâmico e UI
    └── img/            # Assets visuais (Logos, Mascotes e Imagens da Galeria)
        ├── LEGACY_COMMUNITY.png
        ├── LOBO_BRANCO_FUNDO_PRETO.png
        └── IMG-1.png até IMG-12.png
```

## ✨ O que foi feito?

O projeto conta com diversas funcionalidades implementadas para garantir uma excelente experiência ao usuário:

- **Design Responsivo**: Totalmente adaptado para dispositivos móveis, tablets e desktops.
- **Interface Moderna**: Uso de tons escuros (dark mode) com detalhes em degradê e efeitos de brilho (glow) que remetem à identidade visual do servidor.
- **Páginas Temáticas**:
    - **Home**: Banner principal impactante e chamadas para ação.
    - **Links Sociais**: Página exclusiva com botões estilizados para Instagram, YouTube e Discord, facilitando o engajamento da comunidade.
    - **Galeria**: Layout em grid para exibição de imagens do servidor.
    - **IP do Servidor**: Facilidade para os jogadores copiarem e entrarem no servidor.
- **Componentes de UI**:
    - **Menu de Navegação**: Navegação fluida entre todas as seções do site.
    - **Page Loader**: Sistema de carregamento inicial para uma transição suave.
    - **Footer Unificado**: Rodapé profissional com links rápidos e créditos.
